## 📖 Deskripsi

**E-Wallet Desktop Application** merupakan aplikasi dompet digital berbasis **Java Desktop (Swing)** yang dikembangkan sebagai **Proyek UAS Mata Kuliah Pemrograman 1**.

Aplikasi ini menerapkan konsep **Object-Oriented Programming (OOP)** dengan memanfaatkan **Java Swing** sebagai antarmuka pengguna serta **MySQL** sebagai media penyimpanan data. Sistem dirancang untuk membantu pengguna dalam mengelola akun, saldo, dan transaksi secara lebih mudah dan terstruktur.

---

## ✨ Fitur Utama

- Login Admin dan User
- Registrasi Pengguna
- Dashboard Admin
- Dashboard User
- Manajemen Data Pengguna
- Manajemen Saldo
- Riwayat Transaksi
- Penyimpanan data menggunakan MySQL

## 🚀 Cara Menjalankan

1. Install Java JDK.
2. Install Apache NetBeans IDE.
3. Install XAMPP atau MySQL Server.
4. Jalankan service **MySQL**.
5. Buat database bernama:

```
db_ewallet
```

6. Import file:

```
Database/db_ewallet.sql
```

7. Buka folder **EWallet** menggunakan Apache NetBeans IDE.
8. Pastikan library **MySQL Connector/J** telah ditambahkan.
9. Jalankan aplikasi.
